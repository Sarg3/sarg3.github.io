StudGuard

Summary:
A simple server management tool that features an automatic admin, a logging system, server presets, and much more.

It's also extremely simple to add more presets. You just drag the .cs file you downloaded into the presets folder and bam! The file is loaded when you start up Blockland. More info to come.

SpeedKart Laps

A simple way to create laps for your custom SpeedKart map. You'll need a custom map that goes all the way around. This also includes checkpoints that you can go back to mid-race if you get stuck.
